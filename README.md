# Quran Kareem Website

A responsive Quran web application that allows users to listen to Quran recitations from different reciters, browse available surahs, listen to Quran radio stations, and watch live broadcasts of the Holy Quran and Sunnah channels.

## Features

* Select and listen to Quran reciters.
* Choose different Riwayat (Moshaf versions) for each reciter.
* Browse and play available surahs dynamically.
* Listen to multiple Quran radio stations.
* Live streaming of:

  * Holy Quran Channel
  * Sunnah Channel
* Direct streaming of Cairo Quran Radio.
* Responsive and user-friendly interface.
* Modern Islamic-themed design.
