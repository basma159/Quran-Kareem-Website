## Features

* Select and listen to Quran recitations from different reciters.
* Choose different Riwayat (Moshaf versions).
* Browse and play Quran surahs dynamically.
* Listen to Quran radio stations.
* Watch live Quran and Sunnah TV channels.
* Listen to Cairo Quran Radio.
* Listen to Tafsir Al-Tabari audio explanations.
* Select a surah and play its corresponding Tafsir audio.
* Responsive design for mobile, tablet, and desktop devices.
* Clean and user-friendly Islamic interface.

## Technologies Used

### Front-End

* HTML5
* CSS3
* JavaScript (ES6)

### Styling

* Tailwind CSS
* Responsive Web Design
* Custom CSS

### APIs

* MP3Quran API

  * Reciters
  * Mushafs
  * Surahs
  * Radios
  * Tafsir Audio

* AlQuran Cloud API

  * Quran Text

### Libraries

* HLS.js

  * Live streaming support

## What I Learned

* Consuming and working with multiple APIs.
* Handling audio streaming and playback.
* Dynamic rendering of Quran and Tafsir content.
* Managing user interactions with forms and select menus.
* Building responsive interfaces using Tailwind CSS.
* Integrating live video streams using HLS.js.
